# Pokemon System AQ

A question answer system about pokemon

## Dependencies
```
pip install flask

pip install nltk

pip install spacy 

pip install pandas

pip install openpyxl

pip install levenshtein

pip install -U scikit-learn
```
## How to use
Run the server file
```
.\server.py
```

go to the url specified, by defect:
http://127.0.0.1:5000
